
## 功能简介

WeekCalendar


以星期为单位显示日期，即每个界面只显示一周内7天的日期数据，通过左右滑动来切换下一周或上一周，滚动功能依赖swiper组件，具体项目逻辑在src/view/index/index.js,欢迎批评和指正共同进步；
